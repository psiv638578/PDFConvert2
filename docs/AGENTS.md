## 📌 Цель задания
Обновить внешний вид, структуру кода и поведение графического интерфейса модуля `main_gui.py` в проекте PDFConvertProject. Улучшить читаемость кода, поддержать мультиязычность, оформление и документацию.

## 🧩 Разделение задания

### 🔹 Задание 1: Рефакторинг и структура кода
- Разбивка функций по логическим блокам с заголовочными комментариями в стиле **Markdown**
- Соблюдение стандарта кода **PEP8**
- Упрощение структуры и логики для понимания программистом начального уровня

### 🔹 Задание 2: Дизайн интерфейса
- Обновить интерфейс главного окна на основе макета `Дизайн_главного_окна_2.png`
- Добавить меню **Настройка** с подпунктами:
    - **Язык интерфейса**
        - Русский
        - Українська
        - English
    - **Цветовая схема**
        - Светлая
        - Тёмная
- Добавить **заглушки** для этих пунктов с выводом диалога "В разработке"
- Установить **пиктограмму** `logo_ps.png` в заголовок всех окон (с масштабированием)
- В окне **О программе** применить фон на основе `logo_spp.png` (с масштабированием)

### 🔹 Задание 3: Поддержка мультиязычности и тем
- Подготовка структуры для языковых файлов (`translations/ru.json`, и т.д.)
- Поддержка смены языка интерфейса (на уровне заготовок)
- Подготовка структуры для тем оформления (светлая/тёмная) в виде `.qss`
